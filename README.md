# 🍧 Comercio Açaí - Boutique do Açaí

Interface web moderna, responsiva e elegante desenvolvida para uma **Boutique de Açaí Gourmet**. O projeto funciona como um catálogo digital e institucional, focado em proporcionar uma experiência atraente para o cliente conhecer o cardápio e os produtos premium.

🚀 **Status do Projeto:** Em desenvolvimento.

---

## 📷 Demonstração Visual

### Imagens dos Produtos Premium
O projeto conta com uma apresentação visual caprichada de opções selecionadas para abrir o apetite dos clientes:
* **Açaí Tradicional** & **Açaí com Frutas**
* **Açaí com Granola** & **Açaí com Morango**
* O queridinho da casa: **Açaí com Pistache** 🥑✨

---

## 🛠️ Tecnologias Utilizadas

O site foi construído utilizando as tecnologias fundamentais da web (Front-end Vanilla):

* **HTML5:** Estruturação semântica das páginas de navegação e login.
* **CSS3:** Estilização personalizada, design responsivo e identidade visual da marca.
* **JavaScript (ES6):** Manipulação de DOM e lógica de interatividade.
